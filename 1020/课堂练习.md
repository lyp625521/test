# 问题:

* Home 目录是什么？

1、home目录是用户dao的宿主bai目录，一个用户登录系统，进入后，所处的du位置就是/home。
2、通常用来保存dao用户的文件。

```html
这是系统默认的用户家目录(home directory)。在你新增一个一般使用者账号时， 默认的用户家目录都会规范到这里来。比较重要的是，家目录有两种代号喔：
~：代表目前这个用户的家目录，而
~dmtsai ：则代表 dmtsai 的家目录！
```

*  . 和	.. 以及～是什么？

  ​	**.**  : 当前目录

  ​	**..**  : 上一层目录

    **～**：当前用户的家目录

*  什么是隐藏文件？

  以 . 开头的文件，一般看不到
